# Prog_Pj-Loto
LDE, LMY, CAH

Cahier des charges :
Réaliser un programme complet en C, permettant de faire une analyse (statistique) des numéros de loto que l’on a joué.

Condition :
1)  Le  programme testera la présence ou non, d’un fichier texte. Si ce fichier n’est pas présent, le programme devra demander à l’utilisateur plusieurs informations.
2)  Un mode simulation pour des nombres générés des nombre aléatoire selon les informations du loto sélectionné, le nombre de valeur sera indiqué par l’utilisateur (Optionnelle).
3)  Demander à l’utilisateur quels fichiers, celui-ci veut ouvrir, puis afficher en mode console les informations contenues dans ce fichier.
4)  L’utilisateur aura ensuite plusieurs fonctionnalités à choix :
      • Insérer une nouvelle valeur (data)   
      • Modifier ou Supprimer la dernière valeur (data)  
      • Rechercher la valeur la plus « gagnante »  
      • Rechercher la valeur la moins « gagnante »  
      • Afficher la valeur la plus « gagnante »    
      • Afficher la valeur la moins « gagnante »  
      • Afficher les 6 « meilleures » numéro à jouer  
      • Afficher les 6 « moins bon » numéro à jouer 
      • Afficher le nombre de tirage  
      • Créer un nouveau fichier (option)  
        o Nom du loto  
        o Plages possibles  
        o Valeurs complémentaire s  
      • Quitter le programme
    
Attention : L’affichage des 6 numéros se feront dans l’ordre croissant.
